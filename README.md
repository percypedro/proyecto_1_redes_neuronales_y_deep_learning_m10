# Clasificación de Actividades Humanas con Deep Learning

Este proyecto implementa una Red Neuronal Profunda (DNN) para clasificar actividades humanas utilizando datos de sensores móviles.

## Tecnologías utilizadas
- Python
- TensorFlow / Keras
- PCA
- Jupyter Notebook

## Descripción
Se entrenaron dos modelos:
- Modelo con PCA
- Modelo sin PCA

para evaluar el impacto de la reducción de dimensionalidad en el rendimiento del modelo.
